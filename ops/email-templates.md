## Usage Notes

1. **Replace placeholders:**
   - `[SCHOOL_NAME]` - School name from invoice request
   - `[INVOICE_NUMBER]` - Your invoice reference (e.g., INV-2026-0012 or EDU-SCH-0001)
   - `[DUE_DATE]` - Typically 30 days from invoice date
   - `[YOUR_SITE_URL]` - Your website URL
   - `[GOOGLE_DRIVE_OR_DROPBOX_LINK_TO_ZIP]` - Link to the ZIP file

2. **Invoice numbering format:**
   - Choose one format and stick to it
   - Examples: `INV-2026-0012` (year-based) or `EDU-SCH-0001` (simple running number)
   - Use the same reference in invoice, EFT payment, and PoP upload

3. **ZIP file:**
   - Store one ZIP per package in Google Drive or Dropbox
   - Set sharing to "Anyone with the link can view" (or password protect if preferred)
   - Rotate links quarterly if security is a concern

4. **Timeline expectations:**
   - Invoice sent: Within 1-2 business days of request
   - Delivery: Within 1-2 business days of PoP confirmation
